\chapter{Wünsche an FAS}
\label{Wünsche}

\begin{itemize}

\item \textbf{Bekanntgabe}\\
Per E-Mail, Telefonanruf auf der FAS-Hotline -222 oder persönlich bei einem der Mitarbeiter des FAS-Online Projekts.

\item \textbf{Wunschliste}\\
Die Wünsche werden auf einer Liste gesammelt.

\item \textbf{Aufwandschätzung}\\
Entweder wird der Wunsch gleich in die ToDo-Liste gereiht oder es werden im FAS-Meeting allfällige Fragen geklärt.
Eventuell wird hier auch eine Arbeitsgruppe gebildet, um den Wunsch auszuarbeiten.

\item \textbf{Priorisierung}\\
Die ToDo-Liste wird im FAS-Meeting nach Wichtigkeit priorisiert.

\end{itemize}
