\chapter{FAS-Meeting}
\label{Kapitel FAS_Meeting}

\begin{enumerate}

\item \textbf{Neuerungen}\\
Vorstellung von Änderungen und neuen Features zu Beginn eines Meetings.

\item \textbf{Die nächsten Schritte}\\
Hier wird über die Prioritäten der ausstehenden Wünsche abgestimmt.

\item \textbf{Fragen}\\
Zuletzt werden Fragen zu den Wünschen geklärt und gegebenenfalls eine Arbeitsgruppe gebildet.

\info {Es werden während des Meetings keine neuen Wünsche aufgenommen oder bearbeitet.}
\end{enumerate}
