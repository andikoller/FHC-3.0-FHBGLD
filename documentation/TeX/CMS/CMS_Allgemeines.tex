\section{Childs}
Über den Punkt Childs können die Kindelemente eines Contents zugewiesen werden.
Dadurch wird die Menüstruktur definiert. \\
Ein Content darf Child-Element von mehreren Contents sein. (mehrmals im Menü
vorkommen)\\
\\
\section{Zugriffsrechte}
Über den Punkt Gruppen können Benutzergruppen zu Seiten zugeordnet werden. 
Nur Benutzer welche sich in diesen Gruppen befinden dürfen die Seite betrachten.
Wenn keine Gruppe zugeordnet ist, ist die Seite für alle sichtbar.\\
Es können hier nur Gruppen ausgewählt werden, bei denen das Attribut Content 
auf true gesetzt ist.
