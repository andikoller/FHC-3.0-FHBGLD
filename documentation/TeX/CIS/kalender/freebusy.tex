\chapter{Freebusy}
\label{Freebusy}
Freebusy Informationen des persönlichen LV-Plans eines Users stehen unter foglender URL zur Verfügung:
\\
https://cis.technikum-wien.at/cis/public/freebusy.php/<username>\\
\\
<username> ist durch den Loginnamen zu ersetzten.\\
\\
Die Freebusy URL enthält standardmäßig den persönlichen LV-Plan des Users.
Es gibt die Möglichkeit, diese Freebusy URL zu erweitern, um persönliche Kalender in diese URL mitaufzunehmen. (zB Horde Webmail, Google Kalender, DAViCal, etc)
\section{Zusätzliche Freebusy URLs hinzufügen}
Über den Punkt ''Neuen Eintrag hinzufügen'' können zusätzliche Freebusy URLs hinzugefügt werden.
Alle Einträge werden zu einer gesammelten Freebusy URL vereint.

\begin{figure}[ht]
\begin{center}
\includegraphics[width=1.0\textwidth]{freebusy_neu.png}
\end{center}
\caption{FreeBusy URL hinzufügen}\label{FreeBusy URL hinzufügen}
\end{figure}

\begin{enumerate}
	\item Bezeichnung: Freitext Bezeichnung der URL
	\item Typ: Anbieter der URL
	\item URL: Pfad zur FreeBusy URL die hinzugefügt werden soll (wird in manchen Fällen bei der Auswahl des Typs vorgeschlagen)
	\item Aktiv: Dieses Hackerl muss gesetzt sein, damit die Einträge in die FreeBusy URL aufgenommen werden
\end{enumerate}

Der persönliche LVPlan ist automatisch zur FreeBusy URL zugeordnet und kann nicht entfernt werden.
